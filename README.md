# Anime Style Navigation Site

一个二次元风格的个人导航网站，支持深色/浅色主题切换。

## 特点

- 二次元风格设计
- 深色/浅色主题切换
- 主题选择记忆功能
- 响应式设计
- Cloudflare Pages 托管

## 部署方式

1. Fork 本仓库
2. 在 Cloudflare Pages 中连接您的 GitHub 仓库
3. 部署完成后即可访问

## 自定义

您可以通过修改 `script.js` 中的 `links` 数组来自定义导航链接。
